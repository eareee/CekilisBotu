# 🎉 Çekiliş Botu

## ⚙️ Nasıl kurulur?

Öncelikle `.env_config` dosyasını `.env` olarak yeniden adlandırın ve `.env` içindeki değerleri doldurun:

```
# Bot tokeniniz
TOKEN=

# Bot'un prefixi
PREFIX=

# IDniz
AUTHOR_ID=
```
